Forensic DNA Analysis using Binary Search Trees

This project involves using Binary Search Trees to analyze DNA sequences to identify potential owners of unknown DNA based on a provided DNA database.

Project Overview

The Forensic DNA Analysis project aims to identify potential owners of unknown DNA sequences by comparing them with a provided DNA database...

Key Concepts

  • DNA and STRs: Analysis focuses on specific DNA sections called Short Tandem Repeats (STRs).
  • Binary Search Trees (BSTs) are used for efficient storage and retrieval of DNA profiles.

Implementation

  • Profile Class: Manages DNA profiles including occurrences of various STRs.
  • ForensicAnalysis Class: Contains methods for BST manipulation and STR analysis.

This project enhances understanding of data structures and their applications in bioinformatics.

Conway's Game of Life Simulator

This Java application simulates Conway's Game of Life, demonstrating complex behaviors from simple rules.

Technical Overview

  • Programming Language: Java
  • Dynamic Grid and State Management
  • Generational Updates and Neighborhood Computation

Implementation Details

  • computeNewGrid(): Calculates the next state of the grid.
  • nextGeneration(): Progresses the grid to the subsequent generation.
  • numOfAliveNeighbors(): Counts alive neighbors to apply game rules.

Dealt with edge cases and performance issues, gaining insights into array manipulation and algorithm optimization.

Future Enhancements

  • GUI for dynamic interaction
  • Performance optimization for larger grids
  • Feature additions like state saving/loading and rule adjustments

This project is a testament to the application of theoretical computer science principles in practical, engaging simulations.
